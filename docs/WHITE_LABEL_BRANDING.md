# White-Label Branding

Preview-only by default. Settings: brand name, colors, logo URL preview, public site
title, masked support contact, custom domain, footer, powered-by visibility, terms/privacy URLs.


## Safety
- No logo file uploads (URL/text preview only).
- No DNS config, no SSL issuance (custom domain is a manual checklist).
- Powered-by stays VISIBLE unless white-label is explicitly enabled
  (`RESELLER_PORTAL_ALLOW_WHITE_LABEL=true`) AND configured off.
- Support email/phone masked in API responses.
