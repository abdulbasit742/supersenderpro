# Pilot Conversion Playbook


The conversion advisor combines success + risk + blockers. readyToConvert = success >= 70 AND risk < 40 AND no blocked
required checklist items. Suggested plan upgrades to Pro at success >= 85.


## Follow-up drafts
Draft types include demo confirmation, trial approved, setup reminder, missing info, checklist reminder, trial expiring,
upgrade recommendation, feedback request, bug resolved, thank you, cancellation save. Languages: English, Roman Urdu,
mixed.

## Consent
Marketing-type drafts (upgrade, feedback request, trial expiring, thank-you) are blocked when consent is missing and
`PILOT_OPS_REQUIRE_CONSENT=true`, with an admin review note instead. Nothing is ever sent; drafts only.
