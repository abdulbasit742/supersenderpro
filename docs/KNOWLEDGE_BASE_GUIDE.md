# Knowledge Base Guide

Articles have title, slug, category, summary, body, tags, relatedModules, visibility
(internal / public_safe / admin_only), status, lastReviewedAt.

- 15 default articles are seeded on first run.
- Search is keyword-scored, no external service.
- Public-safe articles power the `/help.html` FAQ.
- Never include secrets, real customer data, internal paths, or tokens.
