# Pilot Feedback & Bug Tracking

Types: bug, feature_request, confusion, pricing_feedback, onboarding_feedback, praise, complaint, support_question.
Statuses: new, triaged, planned, in_progress, resolved, rejected, archived.

## Privacy
Raw private customer messages are NOT stored by default. Only a sanitized `descriptionSafe` preview is kept
(phones/emails masked). A heuristic classifier suggests type + severity with no AI key. Bug triage orders by severity.
