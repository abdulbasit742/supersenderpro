# Group Commerce OS — Advanced Features v2

Three new advanced modules have been added to the Group Commerce OS layer. All features are dry-run safe and mask sensitive data.

## 1. Order Book Tracking
**File:** `lib/groupCommerce/orderBook.js`

Tracks order drafts per group with status lifecycle (draft → confirmed → packed → shipped → delivered → cancelled).

**APIs:**
- `POST /api/group-commerce/groups/:id/orders` — Create a new order draft
- `GET /api/group-commerce/groups/:id/orders` — List all orders for a group
- `PUT /api/group-commerce/groups/:id/orders/:orderId` — Update order status
- `GET /api/group-commerce/groups/:id/orders/summary` — Aggregate stats (total orders, by-status counts, projected revenue)

**Example:**
```json
POST /api/group-commerce/groups/group-123/orders
{ "sku": "SKU-IPH13", "quantity": 2, "buyer": "+923001234567", "unitPrice": 145000 }
```

## 2. Fraud / Scam Risk Scoring
**File:** `lib/groupCommerce/fraudScoring.js`

Scores incoming messages for scam likelihood (0–100) based on risk patterns:
- Advance-payment demands
- Too-good-to-be-true investment claims
- Off-platform contact pressure
- Shortened/suspicious links
- High-risk crypto payment mentions
- Artificial urgency tactics
- Phishing-style claims

**API:**
- `POST /api/group-commerce/groups/:id/score-fraud`
```json
{ "messageText": "URGENT! Pay in advance. Double your money. bit.ly/xyz" }
```
Returns `riskScore`, `riskLevel` (low/medium/high), detected `signals`, and `recommendation`.

## 3. Group Analytics & Daily Digest
**File:** `lib/groupCommerce/analytics.js`

Provides activity statistics and automated daily market digests.

**APIs:**
- `GET /api/group-commerce/groups/:id/analytics` — Event counts by type, seller/buyer message tallies
- `GET /api/group-commerce/groups/:id/daily-digest` — Human-readable market summary with low-stock alerts

**Digest Output:**
```

🗓️ Sat Jun 20 2026
═══════════════════════════
🛍️ Active SKUs: 12
📦 Total stock units: 345

⚠️ Low stock alert: iPhone 13 (2 left), SSD 1TB (1 left)
═══════════════════════════
🤖 Generated by Group Commerce OS (dry-run)
```

## Safety Guarantees
- All endpoints return drafts/analytics only — no live orders, posts, or DB writes.
- Phone numbers and sensitive identifiers are masked in every response.
- Fraud scoring is advisory only; no messages are deleted or hidden by default.
