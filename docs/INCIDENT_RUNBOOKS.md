# Incident Runbooks


Each incident can link to a runbook with symptoms, likely causes, safe checks, manual fix steps, related docs, and
severity. Auto-fix is dry-run suggestion only; nothing executes automatically.


## Included runbooks
WhatsApp session disconnected, channel publishing failed, social token missing, ecommerce provider missing, payment
verification stuck, SaaS tenant past due, voice AI provider missing, AI provider key missing, queue stuck, backup restore
blocked, security scan warning, launch readiness blocked, route not mounted, dashboard page missing, env placeholder
missing, live action enabled, customer 360 privacy warning, flow studio trigger missing.
