 # Pilot Onboarding Checklist

 Each pilot gets a generated checklist across 16 sections: business profile, industry preset, admin contact, WhatsApp, AI
 provider, ecommerce, payment, channel/social, Voice AI, Customer 360, owner briefing, playbooks, compliance, demo
 walkthrough, success criteria, go-live.

 ## Item statuses

not_started, in_progress, waiting_customer, waiting_admin, blocked, completed, skipped. Required items count toward go-
live readiness. Progress = completed / total.

## Blockers
Any item set to `blocked`, or any required item not completed, surfaces in the blocker tracker and lowers conversion
readiness.
