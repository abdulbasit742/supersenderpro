# Reseller Referrals

Referral records track lead → conversion with safe ref/UTM codes. No raw lead PII,
no PII in links, no real affiliate payouts.

Statuses: lead_created, demo_requested, trial_requested, onboarding_started,
active_trial, converted_preview, paid_confirmed_manual, rejected, archived.

Referral links use `utm_source=partner&utm_medium=referral&utm_campaign=CODE&ref=CODE`.
Commission preview is estimated from converted referrals (or SaaS Billing if present).
