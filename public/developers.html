API overview

A redacted, preview-first REST API. Authentication uses an X-API-Key header. See the OpenAPI catalog at /api/developer-portal/openapi.json.

Webhook events (public-safe)

n8n / Zapier-style usage

Subscribe to events like public_funnel.lead_created. Every payload is redacted (no full phone, email, payment refs, or secrets). Deliveries are dry-run/simulated until live mode is explicitly enabled by an admin.

Request developer access

Contact your account admin to provision a developer app. API keys are issued as one-time previews and never shown again.

No private endpoints, secrets, or admin-only routes are exposed on this page.
